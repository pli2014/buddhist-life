## 📖 网站概述

这是一个以佛学智慧和人生启迪为主题的静态网站，专为移动端阅读优化。网站通过五个主题页面，深入浅出地探讨了如何将古老的佛学智慧应用到现代生活中。

**特色**: 融入丰富的佛学视觉元素，包括法轮、莲花、曼陀罗背景等，营造庄重典雅的阅读氛围。

## 🎨 CSS架构说明

### 1. **全局样式 (css/style.css)**
包含所有页面共用的基础样式：
- 基础重置和全局设置
- 容器和布局样式
- 通用组件（引用框、智慧盒、实践盒、故事盒等）
- 响应式设计断点
- 打印样式优化

### 2. **页面专用样式**
每个页面有独立的CSS文件，包含：
- 页面特定的背景渐变色
- 主题色彩方案
- 页面独有的组件样式
- 特殊的布局和视觉效果

### 3. **样式加载顺序**
``html
<link rel="stylesheet" href="../css/style.css">    <!-- 先加载全局样式 -->
<link rel="stylesheet" href="../css/[page].css">   <!-- 再加载页面样式 -->
```

## 📝 技术细节

### 前端架构
- **纯HTML+CSS**：无需后端，易于部署
- **模块化CSS**：全局样式 + 页面专用样式
- **移动优先**：为手机端最佳体验设计
- **语义化标签**：利于SEO和无障碍访问
- **跨浏览器兼容**：支持所有现代浏览器
- **零依赖**：不需要任何框架或库

### CSS组织
- **全局样式**：提取公共组件和布局
- **页面样式**：每个页面独立配色和特殊样式
- **层叠策略**：全局样式优先，页面样式覆盖
- **响应式断点**：600px以下启用移动端布局
- **打印优化**：隐藏导航，优化打印效果

### 性能优化
- **外部CSS**：浏览器可缓存，减少重复加载
- **最小化选择器**：提高渲染效率
- **渐变背景**：使用CSS渐变而非图片
- **无JavaScript**：纯CSS实现所有交互效果
